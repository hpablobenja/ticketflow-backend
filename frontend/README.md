# Proyecto de Prueba Backend + Frontend

Este proyecto se centra en el **backend** como la parte más importante, pero se creó un **frontend sencillo** para poder probar las funcionalidades principales.

## 📌 Backend
- Implementado con **FastAPI**.
- Endpoints para registro de usuarios, autenticación y consulta de perfiles.
- Base de datos PostgreSQL con SQLAlchemy y AsyncSession.
- Manejo de roles mediante Enum (`Customer`, `Organizer`, `Admin`).

## 📌 Frontend
- Construido con **Streamlit**.
- Permite:
  - Registrar nuevos usuarios.
  - Iniciar sesión.
  - Consultar y actualizar perfil.
- Su propósito fue únicamente **probar el backend**, no es un producto final.

## 🚀 Objetivo
El objetivo principal del proyecto fue **validar el backend** y sus endpoints.  
El frontend se desarrolló como herramienta auxiliar para realizar pruebas rápidas y verificar que el flujo de registro, login y consulta de usuarios funcionara correctamente.
